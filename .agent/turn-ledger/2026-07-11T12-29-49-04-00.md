## Finding

```txt
safePath(root, path)
  -> resolve(root, path)
  -> target.startsWith(root)
  -> list/read/write/artifact operation
```

String-prefix membership is not path-segment containment. It also ignores filesystem real paths and symlink destinations.

## Decision

Add `meadow-workspace-path-authority-domain` after the host capability gateway and before additional headless automation. Put reusable containment logic in NexusEngine's core headless editor and keep game-specific allowed-root and artifact policies in this repository.
