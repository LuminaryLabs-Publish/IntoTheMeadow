## Main finding

```txt
browser RAF step producer
browser editor step producer
Node editor step producer
  -> all call the same raw game.tick()
  -> no shared admission policy
  -> no finite-value validation
  -> no monotonic-time validation
  -> no work budget
  -> no request identity or duplicate handling
  -> no typed step result
  -> no required render acknowledgement
```

The Node loop can be non-terminating for `ticks = Infinity`. Fractional, negative and non-finite tick counts do not produce truthful typed results. Non-finite or negative delta values can poison or regress local time. Browser editor ticks can mutate state while RAF remains active and can default the recorded time back to zero.
